How to run .PY


----- Option 1 -----

Run:
Braille_Shades_Art_RUN.desktop


----- Option 2 -----

Right-click on the folder
"Open in terminal"
python3 Braille_Shades_Art.py


----- Option 3 -----

Right-click on Braille_Shades_Art.py
Properties...
Permissions
Check "Allow this file to run as a program"
Run by double-clicking or right-clicking and choosing Terminal

"Allow this file to run as a program" can also be added via the terminal:
Right-click on the folder
"Open in terminal"
Send the following:

chmod +x Braille_Shades_Art.py
